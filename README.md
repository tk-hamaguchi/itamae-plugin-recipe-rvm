Itamae::Plugin::Recipe::Rvm
========

Itamae plugin for Setup Rvm.

Installation
--------

Add this line to your application's Gemfile:

```ruby
gem 'itamae-plugin-recipe-rvm', github: 'tk-hamaguchi/itamae-plugin-recipe-rvm'
```

And then execute:

    $ bundle


Usage
--------

Include your recipe.

```ruby
include_recipe 'rvm'
```
